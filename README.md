# T4 English 🌍📘

Projeto de **site institucional e ambiente educacional online** desenvolvido para divulgação e apoio ao curso **T4 English**, com foco no ensino da língua inglesa, apresentação da metodologia e disponibilização de uma **Sala Online** para acompanhamento das aulas.

O site foi desenvolvido utilizando **HTML, CSS e JavaScript**, com atenção à **acessibilidade**, **organização do conteúdo educacional** e **experiência do usuário**, e está hospedado por meio do **GitHub Pages**.

---

👉 https://leticiatorquato.github.io/projeto_final_di/

## 📌 Funcionalidades

### 🌐 Site institucional
- Apresentação do curso **T4 English**
- Seções informativas:
  - Quem sou eu
  - Metodologia
  - Feedbacks em formato de carrossel
  - Como funciona
  - Contato e redes sociais
- Página de **login**
- Página de **inscrição** para novos alunos

### 🖥️ Sala Online
Ambiente virtual de apoio aos alunos, contendo:
- Avisos importantes organizados em formato de carrossel
- Links para aulas ao vivo
- Materiais em PDF organizados por aula
- Links para aulas gravadas
- Lista de atividades com marcação de conclusão
- Lista de avaliações com marcação de conclusão
- Área de treino fonético
- Navegação simples e organizada por seções

---

## ♿ Acessibilidade

O projeto conta com recursos voltados à acessibilidade, como:
- Botão para alternar contraste (modo claro/escuro)
- Botão para aumentar o tamanho da fonte
- Uso de atributos `aria-label` em botões
- Estrutura semântica do HTML
- Textos alternativos (`alt`) em imagens
- Interface clara, legível e organizada

---

## 🛠️ Tecnologias utilizadas

- **HTML5** – Estrutura das páginas  
- **CSS3** – Estilização e layout  
- **JavaScript** – Interatividade, carrosséis e recursos de acessibilidade  
- **GitHub Pages** – Hospedagem do site
